# My Telegram Bot Library for Go

Библиотека для упрощённого создания Telegram-ботов на языке Go.

---

Example .env default:

```
TOKEN_API
ACCESS_CODE
ALLOWED_USER_ID
STATUS_DEBUG=(true/false)
STATUS_CHECK_ACCESS=(true/false)
TZ=UTC
```
